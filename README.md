# Alphalyzer

> 회계사를 위한 AI 데이터 분석 솔루션

Alphalyzer는 회계 및 감사 업무에서 대용량 데이터를 더 빠르고 체계적으로 분석할 수 있도록 지원하는 데이터 분석 도구입니다.  
Excel의 한계를 넘어, AI 기반 분석과 자동화 기능을 통해 회계사의 판단과 검토 업무를 보조합니다.

---

## 소개

회계 업무에서 다루는 데이터는 점점 더 커지고 복잡해지고 있습니다.  
Alphalyzer는 대용량 데이터 처리, AI Assistant, 감사 지원 분석 모듈, RPA 기반 조회 자동화 기능을 통해 회계사의 디지털 업무 역량을 높이는 것을 목표로 합니다.

기존 스프레드시트 중심의 작업 환경에서 벗어나, 더 큰 데이터를 더 안정적으로 처리하고 반복적인 작업을 자동화할 수 있습니다.

---

## Why Alphalyzer?

### 1. 디지털 역량 강화

회계사의 전문 지식에 AI와 데이터 분석 기술을 결합하여, 실무자가 직접 데이터를 탐색하고 분석할 수 있는 기반을 제공합니다.

### 2. 업무 효율 향상

반복적인 데이터 조회, 검증, 정리 작업을 자동화하여 회계사가 더 중요한 판단 업무에 집중할 수 있도록 돕습니다.

### 3. 신뢰 가능한 공통 인프라

표준화된 분석 기능과 업무 흐름을 제공하여, 회계 및 감사 업무에서 일관성 있는 분석 환경을 구축할 수 있습니다.

---

## 주요 기능

### 01. 대용량 데이터 분석

수십만 건에서 수백만 건 규모의 데이터를 처리할 수 있도록 설계되었습니다.  
Excel의 행 제한으로 처리하기 어려운 SAP 원장, 거래 데이터, 전표 데이터 등도 분석 대상으로 활용할 수 있습니다.

### 02. AI Assistant

자연어로 분석 요청을 입력하면 AI가 Python 기반 분석 코드를 생성하여 데이터 분석을 지원합니다.  
업무 담당자는 복잡한 코드를 직접 작성하지 않아도 분석 흐름을 설계하고 결과를 확인할 수 있습니다.

---

## 03. 회계감사 지원 모듈

Alphalyzer는 회계감사 실무에서 자주 활용되는 분석 절차를 모듈 형태로 제공합니다.  
반복적으로 수행되는 감사 데이터 분석을 표준화하여, 감사인이 보다 빠르고 일관성 있게 위험 요소를 식별할 수 있도록 지원합니다.

### 주요 기능

- Journal Entry Test
- 벤포드 분석
- 중복 데이터 탐지
- 누락 데이터 탐지
- 비정상 거래 식별
- 데이터 정합성 검증
- 계정별·거래처별 이상 항목 분석

### 활용 예시

- 공휴일 또는 비업무시간에 입력된 전표 식별
- 마감일 전후 집중 입력된 전표 확인
- 특정 사용자 또는 특정 계정에 집중된 거래 탐지
- 비정상 금액, 반복 금액, 라운드 금액 거래 확인
- 전기 대비 급증하거나 급감한 계정 분석

---

## 04. RPA 및 조서 관리

Alphalyzer는 반복적인 외부 조회 업무와 분석 결과 관리를 자동화할 수 있도록 RPA 및 조서 관리 기능을 지원합니다.  
감사인이 수작업으로 반복 수행하던 조회, 비교, 정리 업무를 줄이고 분석 결과를 체계적으로 보관할 수 있습니다.

### 주요 기능

- DART 공시 조회 자동화
- 사업자 상태 조회
- 외부 데이터 수집 자동화
- 분석 이력 관리
- 감사조서 작성 지원
- 분석 결과 저장 및 재사용
- 감사조서와 분석 결과의 연계 관리

### 활용 예시

- 거래처 사업자 상태 일괄 확인
- DART 공시자료 자동 조회
- 분석 수행 내역 저장
- 감사조서 작성 시 분석 결과 첨부
- 반복 조회 업무 자동화

---

## 실무 활용 사례

Alphalyzer는 회계감사, 재무 분석, 내부통제 점검 등 다양한 실무 환경에서 활용할 수 있습니다.  
대용량 데이터를 기반으로 이상 거래를 탐지하고, 반복적인 분석 절차를 자동화하여 업무 효율을 높일 수 있습니다.

### 1. Journal Entry Test

분개장 데이터를 기반으로 감사상 위험이 높은 전표를 식별합니다.

활용 예시:

- 공휴일 입력 전표 분석
- 비업무시간 입력 전표 분석
- 마감일 전후 입력 전표 분석
- 특정 사용자 집중 입력 전표 확인
- 비정상 상대계정 조합 탐지

### 2. 매출채권 감사

매출채권 데이터와 거래처 정보를 결합하여 회수 가능성과 거래처 상태를 검토합니다.

활용 예시:

- 장기 미회수 채권 식별
- 휴업 또는 폐업 거래처 확인
- 특정 거래처 매출 집중도 분석
- 대손위험이 높은 채권 탐지
- 거래처별 잔액 변동 분석

### 3. 재고자산 분석

재고수불부와 입출고 데이터를 분석하여 재고 흐름과 기말 재고금액의 적정성을 검토합니다.

활용 예시:

- 재고 입출고 흐름 검토
- 이동평균법 기반 기말 재고금액 계산
- 음수 재고 발생 여부 확인
- 장기 미출고 재고 식별
- 비정상 단가 변동 분석

### 4. 데이터 정합성 검증

서로 다른 원천 데이터 간의 일치 여부를 검토하여 누락, 중복, 불일치 항목을 식별합니다.

활용 예시:

- 판매 데이터와 출고 데이터 비교
- 출고 데이터와 입금 데이터 비교
- 3-way match 기반 검증
- 중복 거래 탐지
- 누락 거래 탐지

### 5. 내부통제 점검

업무 프로세스상 통제 위험이 높은 거래나 승인 흐름을 분석합니다.

활용 예시:

- 승인자와 입력자가 동일한 거래 확인
- 특정 사용자에게 집중된 거래 분석
- 비정상 승인 시간대 거래 식별
- 권한 분리 위반 가능성 탐지
- 반복 수정 또는 취소 거래 확인

---

## 이용 안내

Alphalyzer는 회계 및 감사 업무에 필요한 데이터 분석 환경을 제공하기 위해 배포됩니다.  
사용자는 신청 및 라이선스 발급 절차를 거쳐 Alphalyzer를 이용할 수 있습니다.

### 제공 대상

- 한국공인회계사회 등록 공인회계사
- 회비 완납 회원
- Alphalyzer 이용 요건을 충족한 사용자

### 신청 방법

1. 한국공인회계사회 홈페이지에 접속합니다.
2. 회원 계정으로 로그인합니다.
3. Alphalyzer 이용 신청 메뉴로 이동합니다.
4. 이용 신청서를 제출합니다.
5. 승인 후 라이선스를 발급받습니다.
6. GitHub Releases 또는 안내된 다운로드 경로에서 설치 파일을 다운로드합니다.

### 이용 절차

```text
이용 신청 → 승인 확인 → 라이선스 발급 → 프로그램 다운로드 → 설치 및 실행
